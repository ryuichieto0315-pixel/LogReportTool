## 実行方法
1. このリポジトリを clone
2. LogReportTool.sln を Visual Studio で開く
3. ▶ 実行
4. 起動後、読み込みたい CSV ファイルの絶対パスを入力
5. カウントしたい列（1.日付 / 2.ユーザー / 3.操作）を選択
6. 結果がコンソールに表示されます
7. 別のファイルを読み込みたい場合は、案内に従ってパスを入力
8. アプリを終了する場合は「5」を入力

## 開発環境
- Visual Studio 2026
- .NET 10

## 概要
研修で作成した C# 製のコンソールアプリです。  
CSV形式のログファイルを読み込み、「日付」「ユーザー」「操作」の件数をカウントすることができます。  
オブジェクト指向（責務分離・カプセル化）を意識して設計しています。

### CSVサンプル
- `data` フォルダにサンプルファイルを格納
- CSVは以下の形式を想定しています：

Date,UserId,Operation\
2025-12-29,user1,Login\
2025-12-29,user2,Logout\
2025-12-29,user1,Upload
